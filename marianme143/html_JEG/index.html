Platforms

Computer game

----

A type of video game played on a personal computer rather than a video game console or arcade machine.

Home console

----

A video game console that is designed to be connected to a display device, such as a television, and an external power source as to play video games

Handheld console

----

A small, portable self-contained video game console with a built-in screen, game controls and speakers.

Arcade game

----

A coin-operated entertainment machine typically installed in public businesses such as restaurants, bars and amusement arcade.

Browser game

----

A video game that is played via the World Wide Web using a web browser. Browser games can be run using standard web technologies or browser plug-ins.

Mobile game

----

A video game that is played on a mobile phone (feature phone or smartphone), tablet, smartwatch, PDA, portable media player or graphing calculator.

Cloud gaming

----

A type of online gaming that runs video games on remote servers and streams them directly to a user's device, or more colloquially, playing a game remotely from a cloud.

Virtual reality

----

A simulated experience that can be similar to or completely different from the real world.

Emulation

----

An emulator enables games from a console or otherwise different system to be run in a type of virtual machine on a modern system, simulating the hardware of the original and allows old games to be played.
